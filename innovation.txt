# Innovation 
## 1. Dependencies

- Required libraries are listed in [requirements.txt](requirements.txt)
- When the program is executed, a function *ensure_packages()* verifies if the required libraries are installed, and if not, installs them. 
**However, it is assumed that the databases are already saved in the machine**. 

## 2. Helper functions 
Helper functions are created in my_sql_operations.py to make the code of the menu functions easier to read and less repetitive - instead of writing the code inside the menu functions. 
List of helper functions: 
- get_valid_comapany_id()
- get_company_name(company_id)
- get_attendee_name(attendee_id)
- attendee_exists(attendee_id)

## 3. Add new attendee: Neo4j implementation
The menu function to "Add new attendee" adds the attendee to both databases, not just Neo4j. 

## 3. Exceptions 
Throughout the code, generic Exceptions are used to catch all errors, also when not required. 

## 4. database
The folder data is created to save the databases and information on the schemas.
